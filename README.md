# 🧩 **Busca Letras** - Jogo de Caça-Palavras

**Busca Letras** é um jogo de caça-palavras interativo e educativo para todas as idades, desenvolvido com o Flutter. O objetivo do jogo é encontrar as palavras ocultas em uma grade de letras dispostas em várias direções. O aplicativo oferece diferentes modos de jogo para uma experiência divertida e desafiadora.

---

## 🎮 **Funcionalidades**

- **Modos de Jogo**:
    - **Clássico**: Encontre todas as palavras no menor tempo possível.
    - **Tempo**: Encontre o máximo de palavras antes do tempo acabar.
    - **Temático**: As palavras seguem um tema específico, como "animais", "esportes", etc.

- **Grelha de Jogo**:
    - O jogo possui uma grade de letras, onde o jogador pode procurar por palavras na horizontal, vertical e diagonal.
    - As palavras são inseridas em posições aleatórias, criando um desafio dinâmico.

- **Interatividade**:
    - Os jogadores podem arrastar para selecionar as palavras e, ao fazer isso corretamente, as palavras são destacadas automaticamente.

- **UI Personalizada**:
    - A interface é limpa e intuitiva, com uma `AppBar` personalizada e botões que ajudam a navegação no jogo.
    - Suporte para múltiplos modos de navegação e visualização das palavras encontradas.

---

- **Flutter**: Framework utilizado para o desenvolvimento do aplicativo.
